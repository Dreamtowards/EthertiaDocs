# Ethertia

《Ethertia · 以太效应》是一款注重自由创造和探索的沙盒生存游戏，玩家可以在无限大小的程序生成的3D世界中建造、采掘和交互。

得益于激进的体素系统, 世界中的一草一木都可以被采掘或合成。你可以自由组合世界上的资源和物品进行建造, 创造出属于自己的理想世界。

支持多人联机互动, 加入开放的游戏世界中与玩家一起冒险。还可以使用模组扩展游戏功能, 打造更丰富的游戏体验。
